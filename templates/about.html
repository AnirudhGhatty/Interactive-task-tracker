About this project

This project is a Productivity Dashboard built using HTML, CSS, Bootstrap, and JavaScript. It helps students track their daily tasks and monitor their productivity in real-time. Keep a track of your tasks with a live progress bar and automatically calculated statistics and set goals based on your given tasks.


Features

  • Task tracking using interactive checkboxes
  • Automatic calculation of completed and pending tasks
  • Dynamic productivity percentage
  • Visual progress bar
  • Insights based on task completion

How It Works

Users can mark tasks as completed, and the system automatically updates the statistics and progress bar. The insights section provides feedback based on the user's productivity level.
